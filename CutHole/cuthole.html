Start with an empty model.

Draw the geometry that represents the component, i.e. the window frame.

NOTE: The insertion point will be the origin point. The component will be align in the X/Y plane. Look at existing CutHole components to learn how they are drawn.

Once the geometry is drawn, click the button below to group the component geometry. A shortcut can be assigned to this step. The shortcut command is CutHoles.DefineComponentGeometry.

Draw the cutting geometry and then select the geometry that will cut into the geometry that intersects the selection. A shortcut can be assigned to this step. The shortcut command is CutHoles.MakeCuttingObject.

Now save this new model into any content library directory. You will then be able to refresh the content library panel and have your new comonent show up.

The next step is to place CutHoles components on other geometry, like walls. Note that the CutHoles components will not cut into other Groups, just geometry like Faces and Edges. Once the new compoents are placed where they should cut a hole, click the Cut Holes toolbar button or the button below. A shortcut can be assigned to this step. The shortcut command is CutHoles.CutHoles.

Help

There are two parts to this plugin. One part is for designing Hole Cutting Components. This is the first tab in this dialog.

The other part is using the hole cutting components. The hole cutting components show up in the Content Library.
