Generate String Lights

  • Select a line or an arc to generate string lights along
  • Specify fixture and catenary arc options, then click "Generate String Lights"
  • Fixtures will be generated in identical Groups, so manual changes can be applied consistently to all fixtures

Fixture properties:

Number of Fixtures
Vertical Cable or Housing Height
Vertical Cable or Housing Radius
Bulb Radius
Bulbs Per Fixture

Catenary curve properties:

Number of Facets in Curve
Catenary Cable Radius

If creating a new arc:

Cable Dip
