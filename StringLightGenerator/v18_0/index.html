Generate String Lights

  • Select a line or an arc to generate string lights along
  • Set the desired options, then click "Generate String Lights"
  • Fixtures will be generated in identical Groups, so changes in one fixture Group (like applying Materials) will apply to all fixtures

Fixture properties:

Number of Fixtures
Vertical Cable or Housing Height
Vertical Cable or Housing Radius
Bulb Radius
Bulbs Per Fixture

Catenary curve properties:

Number of Facets in Curve
Catenary Cable Radius

If creating a new arc:

Cable Dip
